# 🤖 AI Chatbot - Perguntas e Respostas

### 🚀 Sobre o Projeto
Este projeto consiste em um Chatbot inteligente desenvolvido no **Google Colab**. Ele utiliza técnicas de **Engenharia de Prompt** para fornecer respostas precisas e contextualizadas, demonstrando a integração entre lógica de programação em **Python** e modelos de linguagem (LLMs) via **Groq Cloud**.

O objetivo principal foi criar uma interface de diálogo simples onde o usuário pode interagir com a IA para sanar dúvidas de forma dinâmica.

---

### 🧠 Engenharia de Prompt (Destaque)
Como parte dos meus estudos, este Chatbot foi configurado com:
- **System Prompting**: Instruções específicas para definir a personalidade e o tom de voz da IA.
- **Few-Shot Prompting**: Exemplos inseridos no contexto para melhorar a assertividade das respostas.
- **Tratamento de Saída**: Filtros para garantir que as respostas sejam claras e úteis.

---

### 🛠️ Tecnologias Utilizadas
- **Python**: Linguagem base para a lógica do bot.
- **Google Colab**: Ambiente de desenvolvimento e execução.
- **Groq API**: Infraestrutura de inferência de alta velocidade para LLMs.
- **Markdown**: Para documentação e formatação das interações.

### 🔑 Configuração Obrigatória: Groq API Key

Para interagir com este chatbot, você precisa de uma chave de API gratuita:

1.  Acesse o [Groq Cloud Console](https://console.groq.com/).
2.  Realize o login ou crie uma conta.
3.  No menu lateral, clique em **API Keys**.
4.  Clique em **"Create API Key"**, nomeie-a e **copie o código gerado**.
5.  Esta chave deverá ser inserida no campo indicado dentro do notebook no Google Colab.
